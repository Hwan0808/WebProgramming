[뉴스] Ryzen 5 4400G, Ryzen 3 4200G 성능 유출

작성자 포시포시 20-05-28 18:34 조회 3,894 회

APISAK을 통해 AMD의 차세대 APU, 르누아르 데스크탑 제품에 대한 3DMark 벤치 자료가 공개되었습니다. 공개된 제품은 리테일 제품은 아니고 PRO 시리즈로, 주로 B2B쪽에 사용되는 CPU입니다.

공개된 CPU는 Ryzen 5 PRO 4400G와 Ryzen 3 PRO 4200G입니다.

먼저 Ryzen 5 PRO 4400G입니다.

- 6코어 12쓰레드

- 베이스 3.7GHz

- 부스트 4.3GHz

- GPU 1900MHz (루머로는 CU 7개)

- AM4 소켓

- (루머) TDP 65W

벤치마크는 3DMark Fire Strike로 진행되었습니다. 해당 CPU의 경우 피직스 19113점, 그래픽 4033점을 기록하였습니다.

전세대 리테일에 해당하는 Ryzen 5 3400G의 경우 피직스 12828점, 그래픽 3964점을 기록하였습니다.

Ryzen 3 PRO 4200G 입니다.

- 4코어 8쓰레드

- 베이스 3.8GHz

- 부스트 4.1GHz

- GPU 1700MHz (루머로는 CU 6개)

- AM4 소켓

- (루머) TDP 65W

4200G의 경우 3DMark 11로 벤치마크가 진행되었습니다. 점수는 피직스 6825점, 그래픽 3489점입니다.
